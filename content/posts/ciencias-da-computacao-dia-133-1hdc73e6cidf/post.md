### DELETE table

-   se em uma query DELETE há mais de uma tabela (usando um JOIN, por
    exemplo) você deve especificar qual tabela você quer deletar (caso
    seja apenas uma tabela para ser deletada) fazendo:

``` {code-block-mode="2" spellcheck="false" code-block-lang="sql"}
DELETE table FROM table ....
```
