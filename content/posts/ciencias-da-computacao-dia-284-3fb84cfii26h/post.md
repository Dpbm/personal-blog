#### AWS

-   Na AWS tudo parte de uma VPC(rede virtual)
-   cada VPC pode estar associada a uma região e cada região a uma
    subrede
-   toda VPC pode ser publica, sendo disponibilizada para todo mundo, ou
    privada, um IP interno usado para coisas internas da aplicação, como
    banco de dados etc
-   0.0.0.0/0 é o default gateway
-   tabela de rotas interligam subredes
-   ao derrubar um server da cloud, o DNA ainda funciona

#### TTL

-   é o tempo em que um pacote vai ficar vivo(usado para DNS, CDN, etc.)
