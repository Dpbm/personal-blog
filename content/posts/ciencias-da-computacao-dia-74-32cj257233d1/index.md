Após um longo período de férias, voltamos a programação normal. Agora no
terceiro semestre, abrimos com chave de ouro, Engenharia de Software I.
Aqui vão alguns termos que anotei.








Linked Data → a capacidade de interconectar dados de uma forma que uma
máquina consiga entender

Ontologia → grafo que explica o que cada coisa é

UML → linguagem de modelagem

ciclo em cascata → pega-se todos os requisitos para um projeto,
implementa tudo e entrega tudo

ciclo iterativo → pega-se parte dos requisitos, implementa, entrega, e
continua fazendo isso até todos os requisitos serem feitos

engenheiro de software → projeta e se preciso implementa, além disso
acompanha a implementação e implantação
