






#### Sistemas de informação

-   qualquer sistema (digital ou não) que armazena, coleta, processa e
    distribui informações para a tomada de decisão.

![elementos de um sistema de informação](/image/image_311.png)

-   O ambiente pode ser visto como o local onde o sistema está rodando
-   o feedback pode ser visto como o estímulo de retorno do usuário
-   benefícios → aumento da eficiência, monitoramento, coleta de
    informações, etc.
-   componente do SI → pessoas, hardware, software, dados, rede
-   tipos de usuários do SI → especialista (quem conhece as Tecnologias)
    e usuário final (quem não sabe nada de tecnologia, mas precisa usar
    o sistema)
-   Pessoas de culturas diferentes usam os sistemas de maneiras
    diferentes








### Dimensões do sistema de informação

#### organização

-   como a empresa está organizada
-   Para isso existem 2 tipos de sistemas: sob medida para a empresa e o
    de prateleira (comprado no mercado e a empresa precisa se moldar a
    ele)

#### Tecnologia

-   Tudo envolvendo tecnologia dentro da empresa, desde software ate
    hardware

#### Pessoas

-   tudo envolvendo as pessoas do projeto
-   cultura, organização, etc.








#### Notas

-   SAP é um ERP
-   Monolito → sistema funciona como um bloco único
-   Microserviços → sistema quebrado em pequenos serviços realizando uma
    única função, possuindo seus próprios recursos, como Banco de dados
