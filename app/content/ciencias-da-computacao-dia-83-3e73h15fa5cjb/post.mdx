




### Ciências da computação dia 83

POO --- JAVA

cada comando *case* precisa de um *break*, se não, todos os comandos
case a baixo do que for verdadeiro serão executados.








o comando *continue* ele termina uma iteração e vai para a próxima,
diferente do *break*, que quebra o loop








em Java, precisamos pré-alocar espaço para um array

como em

``` {code-block-mode="2" spellcheck="false" code-block-lang="java"}
int[] a = new int[99];
```

ou

``` {code-block-mode="2" spellcheck="false" code-block-lang="java"}
int[] a = {1, 2, 3};
```

Além disso, não é possível acessar espaços além do alocado para o array,
diferente de C.

Por fim, essa mesma sintaxe pode ser usada para a criação de matrizes

``` {code-block-mode="2" spellcheck="false" code-block-lang="java"}
int[][] a = new int[9][9];
```








Strings em Java são classes com vários métodos dentro, como:

.length() → pega o tamanho da String

.charAt(posição) → pega um caractere de uma dada posição da String

.equals(string) → compara uma string com outra
