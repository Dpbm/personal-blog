engenharia de software

#### Estratégias de teste

-   teste de unidade → testa a menor unidade do software (classes,
    funções, etc) de forma independente
-   teste de integração → testa a comunicação entre as partes (entre
    classes, por exemplo)
-   teste de validação → testa os requisitos
-   teste de sistema → coloca o projeto em um sistema próximo do real e
    testa-se , por exemplo, requisitos não funcionais
