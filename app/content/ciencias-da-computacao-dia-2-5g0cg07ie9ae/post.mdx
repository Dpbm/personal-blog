### (um pedaço)História da computação em ordem

-   Pessoas começaram a fazer contas para a troca de mercadorias
-   Passaram a escrever e fazer cálculos usando a base decimal
-   Criaram o ábaco para cálculos mais rápidos e fáceis (300 A.C.)
-   Primeira calculadora (em 1642 criada por Blaise Pascal da qual se
    chamava Pascaline)
-   calculo do censo dos EUA (1890, pela atual IBM)
-   calculo da trajetória de misseis (ENIAC, 1945)








### **Conceitos passados**

Computador → máquina repleta de circuitos lógicos digitais que executam
programas lógicos

> Hardware + Software = Computador

Segundo o modelo de Von Neumann os programas ficam na memória e assim
que necessário são executados pelo processador e seus periféricos se
encarregam de pedir uma entrada ou mostrar uma saída

> Unidade de controle + unidade lógica aritmética + acumulador = CPU

Unidade lógica aritmética → responsável por cálculos matemáticos e
operações lógicas

Clock

-   é usado para sincronizar todas as operações da CPU
-   a frequência dele significa quantas operações por segundo a CPU pode
    fazer

memória principal → RAM → memória volátil (apaga quando o computador é
desligado)

Kilo → 2¹⁰ → 1024

Mega → 2²⁰

Giga → 2³⁰

Tera → 2⁴⁰

o menor número de uma base → 0

o maior numero de uma base → base --- 1

ex: hexa → 16--1 = 15 (F)
