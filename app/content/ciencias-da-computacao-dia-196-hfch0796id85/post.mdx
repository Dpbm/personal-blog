




### Ciências da computação dia 196

sistemas de informação

#### Tipos de SI

cada tipo de SI tem uma tarefa especifica que é executada

![camadas de uma
organização](https://cdn-images-1.medium.com/max/800/1*r4Kq7HpX0ObxjW9PI82LPg.png)

Transacional → realiza transações (vendas, reservas, folha de pagamento,
etc.)

Especialista → armazena dados que geram conhecimento para a empresa
(como dados de pesquisa de satisfação), podendo então uma AI ser
utilizada aqui para dar insights. a parte operacional da organização
alimenta o sistema enquanto a gerancial usa

Gerenciais → gera relatórios para a gerencia, trabalhando com problemas
estruturados, sendo ferramentas analíticas

Para tomada de decisão → auxilia na tomada de decisão, usando dados dos
outros 3








ERP → sistema que engloba tudo isso em um unico lugar
