estrutura de controle (seleção)(comando if)

-   usado para escolher qual bloco será executado
-   executa algo caso a instrução for verdadeira
-   caso a condição seja falsa ele ignora e continua com que está de
    baixo do bloco do if
-   o que está dentro é dependente da condição

seleção composta (comando, if, elif e else)

-   todo else possui um if correspondente
-   else é a negação da condição do if
-   caso o if for executado, o else será ignorado e o programa partirá
    para o que está abaixo
-   sempre é executado apenas 1 ou o if ou o else
