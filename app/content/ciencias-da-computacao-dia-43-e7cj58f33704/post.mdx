strings

substring → um pedaço de uma string (texto)

slicing → fatiar uma string em substrings

funcão list (em Python), transforma uma string em uma lista de
caracteres

função join (também em Python), pega uma lista de caracteres e junta
tudo em uma string separando cada letra pelo caractere que você passou
para a função join

ex:

c = \["b", "o", "m", " ", "d", "i", "a"\]

"".join(c) → 'bom dia'

"-"join(c) → 'b-o-m- -d-i-a'
