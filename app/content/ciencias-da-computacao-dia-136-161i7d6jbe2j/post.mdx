Detalhes sobre classes abstratas e interfaces --- POO

-   Java Permite você usar o extends e o implements ao mesmo tempo;
-   É Possível definir uma variável com o tipo referente a interface
    (isso é útil quando a subclasse não tem métodos específicos);
-   Pense em usar classes abstratas quando você precisa definir métodos
    concreto e/ou possui atributos.
